Sistema Inteligente de Gestión de Inventarios

Este proyecto es un sistema desarrollado para optimizar la gestión de inventarios en empresas minoristas mediante técnicas de análisis de datos y Machine Learning. Proporciona herramientas que permiten explorar datos históricos, predecir la demanda de productos, optimizar los niveles de inventario y generar alertas para productos con bajo stock.

El sistema cuenta con una interfaz gráfica desarrollada en Tkinter, que facilita al usuario la ejecución de las funcionalidades principales de manera intuitiva. Entre sus capacidades clave se encuentran:

    Exploración de Datos: Realiza un análisis descriptivo y visualiza patrones de ventas por categorías, mostrando estadísticas detalladas de los productos.
    Predicción de Demanda: Utiliza redes neuronales implementadas con TensorFlow para anticipar las ventas futuras, basándose en variables como cantidad, precio y fecha.
    Optimización de Inventarios: Calcula niveles óptimos de stock mediante un análisis automatizado, considerando un buffer ajustable.
    Generación de Alertas: Detecta productos con inventarios bajos y genera reportes claros y accionables.

La tecnología utilizada incluye Python 3.12, Pandas, Matplotlib, TensorFlow y Scikit-learn, lo que garantiza un rendimiento robusto y resultados confiables. El proyecto organiza los datos en módulos independientes para exploración, predicción, optimización y generación de alertas, lo que lo hace altamente escalable y modular.

Con este sistema, las empresas pueden tomar decisiones informadas para evitar desabastecimientos, reducir costos asociados con exceso de inventario y anticiparse a las demandas del mercado. Es ideal para implementaciones en el sector retail, pero puede adaptarse a otros entornos comerciales.

Para utilizarlo, simplemente instala las dependencias, carga los datos en la carpeta correspondiente y ejecuta el sistema desde su interfaz gráfica. Los resultados generados incluyen visualizaciones, predicciones y reportes exportables en formato CSV. Este proyecto es una herramienta valiosa para quienes buscan transformar la gestión de inventarios con inteligencia y precisión.
